1.	Go to the directory \xampp\htdocs\

	Clone the repository

	$ git clone https://github.com/elizabethordonez/finalproject.git


2.	Repository Directories:
	Database - Initial tables creation and load data
	Documents - Documents related with the project: Project proposal, ER Diagram, group members activity logs.
	finalproject - All files needed to run the web site Visa Process 

	The finalproject repository will be located under the directory \xampp\htdocs\finalproject
	The web site project will be located under the directory \xampp\htdocs\finalproject\finalproject

3.	Open XAMPP Control Panel Session
	Start Apache and MySQL

4.	Open a phpMyAdmin session 

5.	Create the database and tables
	In MySQL create a database called 'visaprocess'
	Click Import button and choose file visaprocess_DDL.sql located in \xampp\htdocs\finalproject\finalproject\ then click Go

6. 	Edit the config.php file located in \xampp\htdocs\finalproject\finalproject and change update the database credentials. 

5. 	Access the index.php file using a browser:

   	http://localhost/finalproject/finalproject/index.php
